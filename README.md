# Token Viewer 🪙

Interface web Python pour visualiser la tokenisation de phrases avec différents modèles de langage.

## Fonctionnalités

- 🔤 Saisie de phrase libre
- 🔢 Affichage des IDs de tokens
- 🏷️ Affichage des tokens textuels
- 📦 Version compressée (run-length encoding)
- 📊 Statistiques (nombre de tokens, taux de compression)
- 🔄 Sélecteur de modèle de langage (GPT-2, BERT, RoBERTa, XLNet, T5)

## Installation

```bash
pip install -r requirements.txt
```

## Utilisation

```bash
python main.py
```

Puis ouvrir http://127.0.0.1:8000 dans le navigateur.
